Instructions on viewing run files

Download the runs and check them out with Teacup Timer .

To view the run:

  1. Make sure the run(therun.run) is in your \runs directory ( e.g. c:\TCT2.5\runs\therun.run )
  2. Start Teacup Timer
  3. At the Main Menu select the Replay Run function
  4. Enter the filename therun.run (you must type it out, we will impliment a graphical file browser in 3.0)
  5. Select the playback speed setting using the arrow keys
  6. Hit enter to begin playback
